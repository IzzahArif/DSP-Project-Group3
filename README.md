# DSP Project: ECG Signal Refining and Analysis

*Group 3 - Digital Signal Processing*

## Project Overview
This project involves the acquisition and processing of ECG signals. We use Digital Signal Processing (DSP) techniques to remove noise (like powerline interference and baseline wander) to provide a clear signal for analysis.

## Repository Structure
* */Data*: Contains raw ECG signal datasets (CSV/MAT files).
* */Scripts*: Contains the Python/MATLAB code for filtering and refining signals.
* */Results*: Visual plots of raw vs. filtered ECG waves.
